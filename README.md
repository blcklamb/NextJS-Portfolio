## Getting Started

아래 사항을 따라 코드블록을 복사하여 터미널에 실행하고, 브라우저 주소창에 [localhost:3000](localhost:3000)으로 이동하세요.

### 설치

<!-- 수정 필요 -->

1. 레포지토리 클론
   ```sh
   git clone https://github.com/blcklamb/NextJS-Portfolio.git
   ```
2. NPM 패키지 다운로드
   ```sh
   yarn
   ```
3. Next 실행
   ```sh
   yarn dev
   ```
